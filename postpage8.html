Website Design with Figma, HTML, CSS, & JavaScript

Date: February 2025

Background

Context: This web design project was created as part of a studio project, focused on practicing with images, JavaScript, and user interaction.

Role: Web Designer

Project Type: Website Design with Figma, HTML, CSS, and JavaScript

Initial Design and Development

This project began with gathering images and brainstorming ideas for the site. I decided to base it on my first trip to Mexico, showcasing some of the memorable moments I captured during my visit. Once I had a clear concept, I moved on to the coding phase to test the functionality of my idea. This was my initial draft, where I focused on experimenting with JavaScript to make sure it worked with the image divs, overlays, and close buttons. It helped me test some of the features and ensure they were working before further moving forward with development. Here's a short video showcasing that:

Final Design & Development

After creating the initial site and receiving feedback from other designers, I realized that a horizontal layout would be more effective for user interaction and viewing. I also added background color to the image divs on hover to better highlight which section the user is on. Additionally, I included arrows on the image slideshows to signal that there are multiple images to view, making the navigation more intuitive.

And here is the link to the live website now and a video that showcases the site.

Final Thoughts: This project was a good opportunity to practice developing interactive features and working with images and JavaScript. Overall, it was a valuable experience for improving my technical skills in web development.
