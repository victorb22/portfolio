Website Design with HTML & CSS

Date: January 2025

Background

Context: This is web design done for a studio project, the goal was to practice working with CSS animations. Specifically, automatic time-based behavior (using transition, transform, keyframes or a combination) and interactive behavior (on hover or time-based).

Role: Web Designer

Project Type: Website Design with HTML, CSS, and Figma

Low & Mid-Fidelity Designs

I started my idea for this project with a quick sketch. It was going to be a portal page that held links to other sites so I leaned on the idea of a 'portal'. I wanted to have the portal opening and closing around my name and title of the page, as well as include a second animation which made my buttons shake on a mouse interaction.

Final Design & Development

For my automatic animation, I manipulated the opacity and scale to get the effect of the portal opening and closing. I also used rotate and transform to make the button links shake when hovered on.

And here is the link to the live website now and a video that showcases the site.

This website ended up serving as a portal page for the rest of my projects in this coding class. That is why in the final video you can see me open up another assignment which is the Learning Journal where we post about different topics learned in class.

Final Thoughts: This project served as good practice for CSS animations and working on page layouts overall. I think you can do a lot with CSS and animations and will continue to practice it along with JavaScript.
